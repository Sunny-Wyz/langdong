## 需求预测算法对比

| 算法 | 适用场景 | 最优条件 | 性能指标 | 局限性 | 推荐使用 |
|------|---------|---------|---------|--------|---------|
| **RF**<br>随机森林 | 规律型需求<br>(ABC-X类) | ✅ 历史数据≥4个月<br>✅ 非零需求月数≥3<br>✅ 需求相对稳定 | MAE: 2.10<br>MAPE: 10.10%<br>月度胜率: 100% | ⚠️ 样本少时易过拟合<br>⚠️ 滞后1-2阶，对突变反应慢 | ✅ **生产首选** |
| **SBA**<br>平滑算法 | 间断型需求<br>(ABC-Y/Z类) | ✅ ADI > 1.32<br>✅ CV² > 0.49<br>✅ 需求有明显间隔 | 适合波动型 | ⚠️ 参数(α, β)敏感<br>⚠️ 调优需人工介入 | ✅ **生产备选** |
| **FALLBACK**<br>均值法 | 数据严重不足 | ✅ 历史 < 3个月<br>✅ 无规律性特征 | 简单稳健<br>误差 ±50% | ❌ 无法捕捉趋势<br>❌ 精度低<br>❌ 仅用于兜底 | ✅ **兜底方案** |

## 安全库存 (SS) 计算方法对比

| 方法 | 公式 | 适用范围 | 优势 | 劣势 | 项目应用 |
|------|------|--------|--------|--------|---------|
| **动态法**<BR>(推荐) | $SS = z \times \sigma_d \times \sqrt{L}$ | 波动型需求<br>有预测区间 | ✅ 根据波动自适应<br>✅ 库存占用最优<br>✅ 科学透明 | ⚠️ 需要预测区间<br>⚠️ 参数 z 需调优 | ✅ **生产实施** |
| **固定法** | $SS = SS_{fixed}$ | 稳定型需求<br>无历史数据 | ✅ 简单易行<br>✅ 易沟通 | ❌ 无响应波动<br>❌ 覆盖率不确定<br>❌ 可能过度库存 | ⚠️ **备选/初期** |
| **经验系数法** | $SS = \mu_d \times L \times \alpha$ | 中等波动<br>有业务经验 | ✅ 考虑消耗量<br>✅ 业界通用 | ⚠️ 系数 α 主观性强<br>⚠️ 难以量化 | 🟡 **参考方案** |

**系统当前口径**（优化后）：
```
sigma_d = (upper_bound - lower_bound) / (2 × 1.645)  // 90% 置信区间反推
SS = ceil(k × sigma_d × sqrt(leadTime_days))
ROP = ceil(daily_demand × leadTime_days + SS)

其中:
  k(A) = 2.33    # 重要件，99% 服务水平
  k(B) = 1.65    # 普通件，95% 服务水平
  k(C) = 1.28    # 非关键，90% 服务水平
```

## ABC×XYZ 分类指导

**分类目的**：根据备件的金额重要性和消耗规律性，指导库存策略

### 分类维度说明

**ABC 维度**（金额重要性）：
- **A类**：年消耗金额 > 750万（占总金额80%）→ 严格库存　
- **B类**：年消耗金额 150~750万（占总金额15%）→ 中度控制　
- **C类**：年消耗金额 < 150万（占总金额 5%）→ 宽松管理　

**XYZ 维度**（消耗规律性）：
- **X类**（稳定）：CV² < 0.49，需求规律 → 高库存周转
- **Y类**（波动）：0.49 ≤ CV² < 1.0，有季节性 → 中等库存
- **Z类**（不规律）：CV² ≥ 1.0，需求随机 → 保持安全库存

### 9宫格策略矩阵

```
           稳定(X)          波动(Y)         不规律(Z)
高重(A) │ 高周转+低SS  │ 中周转+中SS  │ 低周转+高SS  │
        │ 定期自动补  │ 按需补货    │ 尽量外包    │
        
中重(B) │ 中周转+低SS  │ 中周转+中SS  │ 低周转+中SS  │
        │ 月度补货    │ 按需补货    │ 需求预示    │

低重(C) │ 低周转+宽松  │ 低周转+宽松  │ 低周转+宽松  │
        │ 极少补货    │ 一年一补    │ 考虑直采    │
```

## 实验验证结果

### 验证 1：动态法 vs 固定法（真实数据）

**样本**: SP20001 深沟球轴承 | **期间**: 2025-04 ~ 2025-10 (7个月)

| 指标 | 动态法 | 固定法 |
|------|--------|--------|
| 覆盖率 | 0.00% | 57.14% |
| 平均库存 | 1.57 件 | 10.00 件 |
| 平均缺口 | 9.38 件 | 3.10 件 |

**结论**：在真实低波动数据下，固定法表现更好（覆盖率高、缺口小）  
**原因**：该备件消耗稳定，动态参数过低  
**优化方向**：需要引入预测区间上下界，提高 σ_d 估计准确度

### 验证 2：动态法 vs 固定法（可控造数）

**样本**: 人工构造需求波动增大 | **期间**: 2025-04 ~ 2025-10 (7个月)

| 指标 | 动态法 | 固定法 |
|------|--------|--------|
| 覆盖率 | 100.00% | 57.14% |
| 平均库存 | 13.86 件 | 10.00 件 |
| 平均缺口 | 0.00 件 | 0.71 件 |

**结论**：在波动充分的数据下，动态法完全优于固定法  
**适用性**：季节性、突发型需求场景下效果显著

### 验证 3：RF vs SMA(3) 预测对比

**样本**: SP20001 | **期间**: 2025-07 ~ 2026-02 (8个月测试)

| 指标 | RF | SMA(3) |
|------|-----|---------|
| MAE | 2.15 | 3.67 |
| RMSE | 2.21 | 3.80 |
| MAPE | 10.72% | 18.34% |

**结论**：RF 在所有指标均优于 SMA(3)  
**使用建议**：当历史数据 ≥ 4个月时，优先选用 RF

## 算法选择决策树

```
基于历史消耗数据 12 个月

┌─ 非零需求月数 ?
│
├─ < 3 个月 → FALLBACK (均值法)
│
└─ ≥ 3 个月
   │
   ├─ 计算 ADI = 12 / 非零月数
   ├─ 计算 CV² = (σ / μ)²
   │
   └─ ADI > 1.32 AND CV² > 0.49  ?
      ├─ YES → SBA (平滑法，适合间断型)
      └─ NO  → RF (随机森林，适合规律型)

安全库存计算:
├─ 有预测区间（上下界）?
│  ├─ YES → 动态法 (推荐)
│  └─ NO  → 固定法 (备选)
│
└─ 参数:
   ├─ k(z系数): 按 ABC 分类选择 2.33 / 1.65 / 1.28
   ├─ L (提前期): 从供应商档案读取
   └─ σ_d: 从预测区间反推或历史波动计算
```
