# 深沟球轴承（SP20001）RF vs SMA 实验报告

## 1. 实验目标与样本说明
- 样本备件：`SP20001`（深沟球轴承，型号 `6205-2RS`）。
- 实验目标：基于实际消耗记录，对比
  - 随机森林回归（RF）
  - 简单移动平均法（SMA, 窗口=3）
  在测试集上的预测效果。

## 2. 数据来源与口径
- 数据来源：`biz_requisition_item.out_qty` + `biz_requisition.approve_time` + `spare_part`。
- 过滤条件：
  - `sp.code = 'SP20001'`
  - `r.req_status IN ('OUTBOUND','INSTALLED')`
  - `ri.out_qty > 0`
- 月度聚合口径：按 `DATE_FORMAT(r.approve_time, '%Y-%m')` 汇总。
- 缺失月份处理：补 `0`，保证序列连续。

## 3. 训练/测试划分
- 训练集：`2025-01` ~ `2025-12`（12个月）。
- 测试集：`2026-01` ~ `2026-02`（2个月，按当前数据库可用完整月）。

## 4. RF 方法说明
- 模型：`RandomForestRegressor(n_estimators=200, max_depth=5, random_state=42)`。
- 特征：
  - `lag1`：前1个月实际值
  - `lag2`：前2个月实际值
  - `roll3`：前3个月实际均值
- 训练样本数：9（`2025-04` 到 `2025-12` 对应目标值）。
- 预测方式：测试期单步滚动（rolling one-step）。
  - `2026-01` 用 `2025-10~12` 实际值构造特征。
  - `2026-02` 用 `2025-11, 2025-12, 2026-01(真实)` 构造特征。

## 5. SMA 方法说明
- 方法：`SMA(3)`。
- 规则：当月预测 = 前3个月实际值均值。
- 测试期同样采用单步滚动。

## 7. 指标汇总表
- 指标定义：
  - `MAE = mean(|y - y_hat|)`
  - `RMSE = sqrt(mean((y - y_hat)^2))`
  - `MAPE = mean(|y - y_hat| / y) * 100%`（`actual=0` 的样本不计入）

| model | MAE | RMSE | MAPE |
|---|---:|---:|---:|
| RF | 2.15 | 2.21 | 10.72% |
| SMA(3) | 3.67 | 3.80 | 18.34% |

## 8. 结果分析与结论
- 在当前样本与口径下，RF 在三项指标（MAE、RMSE、MAPE）均优于 SMA(3)。
- 结论：本实验中 RF 为优胜模型。
- 原因（结合样本特征）：
  - 序列存在较明显波动与结构变化，RF 对非线性关系的拟合能力优于固定窗口均值法。
- 限制：
  - 测试集仅 2 个月，统计显著性有限；该结论用于“当前样本演示”是成立的，但不应直接外推到所有备件。

## 9. 复现实验 SQL 与计算步骤（只读）

### 9.1 SQL：提取 SP20001 月度实际消耗
```sql
SELECT DATE_FORMAT(r.approve_time, '%Y-%m') AS month_key,
       SUM(ri.out_qty) AS qty
FROM biz_requisition_item ri
JOIN biz_requisition r ON ri.req_id = r.id
JOIN spare_part sp ON ri.spare_part_id = sp.id
WHERE sp.code = 'SP20001'
  AND r.req_status IN ('OUTBOUND','INSTALLED')
  AND ri.out_qty > 0
  AND r.approve_time >= '2025-01-01'
  AND r.approve_time <  '2026-03-01'
GROUP BY DATE_FORMAT(r.approve_time, '%Y-%m')
ORDER BY month_key;
```

### 9.2 Python 复现要点
1. 将 `2025-01`~`2026-02` 建立完整月轴并补0。
2. 划分：2025训练、2026-01~02测试。
3. RF 特征：`lag1/lag2/roll3`，模型参数固定。
4. RF 与 SMA 都按 rolling one-step 预测。
5. 输出逐月对比表与 MAE/RMSE/MAPE 汇总。
