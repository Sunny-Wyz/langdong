# 基础数据管理系统 - 新增功能验收指南

本系统已新增了“货位档案管理”、“设备档案管理”及“供应商档案管理”（含供货品类字典）等核心功能子模块，均已完整开发并覆盖从数据库建表至前端页面的全链路。

以下是相关功能的验证方式说明：

## 一、货位档案管理（含备件分配）功能
（之前已验收，跳过详述）在 `/home/location-profiles` 查看货位及右侧“查看备件”。也可以在备件创建时选择货位。

---

## 二、设备档案管理（含配套备件多对多关联）功能
（之前已验收，跳过详述）在 `/home/equipment-profiles` 查看设备台账，并点击“配套备件”管理与其多对多绑定的部件关系。后端删除设备时会自动级联取消绑定。

---

## 三、供应商档案管理与供货品类配置

### 1. 系统变更范围
1. **数据库层**: 
   - 新建 `supplier`（供应商档案表）、`supply_category`（供货品类字典表）、以及 `supplier_category_relation`（多对多关联表）。
   - 修改 `spare_part` 表结构，加入 `supplier_id` 外键约束关联具体的供应商表项。
2. **后端层**: 提供 `SupplierController` 与 `SupplyCategoryController`，支持全套 CRUD 及供应商的“商品品类配置”动作。
3. **前端层**:
   - 在左侧菜单栏加入了“**供应商档案管理**”入口 (`/home/supplier-profiles`)。
   - 实现了一个专属的供应商台账界面，该界面具有一键链接至“**品类字典**”的基础配置操作入口。
   - 所有备件界面的“添加/编辑备件”弹窗中，原有的**散文本输入框**已被重构更新为支持模糊匹配的**供应商下拉选择器**。

### 2. 验证步骤
1. **强制刷新您的浏览器页面**以获取左侧菜单栏：“基础数据管理” -> “供应商档案管理”。
2. 进入后，可点击左上方灰色的“**品类字典**”按钮，先建立几个品类 (如：电机类、传动类)。
3. 返回供应商页面，点击“新增供应商”，写入公司名称、开户行、社会信用代码等业务字段并保存。
4. 在刚刚创建的供应商所在行，点击绿色的“**供货品类**”按钮，打开关联选择器，将您刚才建好的几个品类字典项分配给该供应商保存。
5. 切换回左侧主菜单的 **"备件管理"**。
6. “增加备件”或勾选一条现有的备件点击“编辑”。系统表单的左下方，原本只能手动签入名称的位置现在应该会出现一个“请选择供应商”的下拉框。
7. 在这里选择您刚才刚刚创建维护过的那一家合格供应商，并保存备件。查看列表中是否正确带出了该供应商的名字。
